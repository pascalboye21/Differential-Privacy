# Differential-Privacy
Im Rahmen meiner Masterarbeit habe ich den Exponentiellen Mechanismus implementiert, um anhand eines praktischen Beispiels eine Differential-Privacy-Datenbank zu erstellen, die ohne Verletzung der Datenschutzbestimmungen veröffentlicht werden kann. Die Datei „main.py” zeigt, wie ich die beiden anderen Dateien für diesen Zweck verwendet habe. Kommentare sind in main.py nicht enthalten.

As part of my master's thesis, I implemented the exponential mechanism to create a differential privacy database based on a practical example that could be published without violating data protection regulations. The file “main.py” shows how I used the other two files for this purpose. Comments are not included in main.py.
